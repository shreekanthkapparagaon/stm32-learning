# LED Pattern Controller (STM32)

## Concepts

* GPIO output
* Delay timing
* PWM (analogWrite)

## Features

* Slow blink
* Fast blink
* Smooth fade effect

## Hardware

* STM32F103C8T6 (Blue Pill)
* LED + 220Ω resistor

## Pin

* LED → PA5

## How to Run

1. Open project in PlatformIO
2. Connect ST-Link
3. Upload code

## Output

* LED blinks (slow → fast)
* Then fades smoothly
* Repeats continuously
